The easiest solution is based on the Bootstrap Grid, which relies on Flexbox. You define rows which are wrappers for columns. Every row has exectly twelve columns. Every item in a row can span one ore more columns. If you take up too much space, the items will simply wrap as usual with Flexbox. That's it.

In this example every items spans 12 columns. See what happens when you add col-md-3 to every item.
